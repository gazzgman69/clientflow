Rules

Dates in output as dd/mm/yyyy.

Phase A — Read-only verification

DB checks

Count Gmail integrations by tenant_id and verify no NULL:

SELECT tenant_id, COUNT(*) AS cnt
FROM calendar_integrations
WHERE service_type = 'gmail'
GROUP BY tenant_id
ORDER BY cnt DESC;
SELECT COUNT(*) AS null_integrations
FROM calendar_integrations
WHERE service_type = 'gmail' AND tenant_id IS NULL;


Emails table isolation & recent window:

SELECT tenant_id, COUNT(*) AS total
FROM emails
WHERE provider='gmail'
GROUP BY tenant_id;

-- last 30 days
SELECT COUNT(*) AS recent
FROM emails
WHERE provider='gmail'
  AND received_at >= NOW() - INTERVAL '30 days';
