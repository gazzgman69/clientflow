Goal
Create real API routes and a provider dispatcher so the app knows how to:

list available providers

start OAuth (Gmail, Microsoft)

handle callbacks

report connection status

disconnect

send email via the active provider

kick off contacts-only sync

Everything must be tenant-scoped.

Add/fail-fast on:

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_OAUTH_REDIRECT_URL=https://<app-domain>/api/auth/google/gmail/callback

MS_CLIENT_ID=
MS_CLIENT_SECRET=
MS_OAUTH_REDIRECT_URL=https://<app-domain>/api/auth/microsoft/callback
MS_TENANT=common

3) OAuth endpoints
Gmail

GET /api/auth/google/gmail → start OAuth. Build URL with scopes:

https://www.googleapis.com/auth/gmail.modify

https://www.googleapis.com/auth/gmail.send

include state = base64json({tenantId,userId,returnTo})

GET /api/auth/google/gmail/callback → exchange code; encrypt tokens; upsertEmailIntegration({status:'connected', account_email, scopes, expires_at})

GET /api/auth/google/gmail/status → {connected, accountEmail, scopes, lastSyncedAt}

POST /api/auth/google/gmail/disconnect → revoke & status:'disconnected'

Microsoft (Graph)

GET /api/auth/microsoft → start OAuth (PKCE if your auth lib supports it) with scopes:

offline_access, Mail.Read, Mail.Send

GET /api/auth/microsoft/callback → exchange; persist like above

GET /api/auth/microsoft/status

POST /api/auth/microsoft/disconnect

All above behind ensureUserAuth, tenantResolver, requireTenant. Add fail-fast if tenantId missing.

4) Provider dispatcher (send + sync)

Create emailProviderDispatcher.ts:

getActiveProvider(tenantId, userId) -> 'google' | 'microsoft' | null

sendEmail(tenantId, userId, payload) {
  const p = getActiveProvider(...);
  if (p==='google') return gmailService.sendEmail(...);
  if (p==='microsoft') return msService.sendEmail(...);
  throw new Error('NO_ACTIVE_EMAIL_PROVIDER');
}

syncContactsOnly(tenantId, userId) {
  const p = getActiveProvider(...);
  if (p==='google') return gmailService.syncContactsOnly(...);
  if (p==='microsoft') return msService.syncContactsOnly(...);
}


Add route:

POST /api/email/send → uses dispatcher.sendEmail (respects per-tenant “BCC me” / “read receipts” prefs).

POST /api/email/sync → trigger one-off sync via dispatcher.syncContactsOnly.
