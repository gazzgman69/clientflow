1) Schema & Seeding

Create a new table (global, not tenant-scoped): email_provider_catalog

id uuid pk default gen_random_uuid()

code text unique not null // e.g. gmail, office365, icloud, yahoo, godaddy_workspace, cox_business, att, aol, 1and1, hotmail_live, siteground, sky, zoho, etc.

display_name text not null // “Google Gmail”, “Office 365”, …

auth_type text not null // enum: oauth, imap_smtp

imap_host text null

imap_port int null

imap_secure boolean null // TLS true/false

smtp_host text null

smtp_port int null

smtp_secure boolean null

help_blurb text null // short provider-specific tips

is_active boolean default true

created_at timestamptz default now()

updated_at timestamptz default now()
