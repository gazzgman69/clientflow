GOAL
Make every control on **Lead Capture Forms** functional:
- Load list of forms
- “+ Add Method” creates a new form and selects it
- Selecting a form loads its details & questions
- “Save” persists form + questions
- “Preview” opens hosted /f/:slug in a new tab
- “Install Form” menu shows working copyable snippets (Hosted, Embed, Popup)
- “+ Add Question” adds to list; delete works; required/mapTo stored
- Public page /f/:slug renders & submit works (already implemented)

SCOPE (EDIT/CREATE ONLY)
  - server/src/routes/leads.ts                // verify/admin & public endpoints
  - server/src/services/lead.ts              // ensure form CRUD helpers
  - web/src/pages/leads/LeadCaptureBuilder.tsx
  - web/src/components/leads/InstallFormMenu.tsx
  - web/src/components/leads/QuestionEditor.tsx
  - web/src/pages/public/LeadFormHosted.tsx  // ensure supports ?embed=1 & ?dialog=1
  - web/src/router.tsx (or web/src/App.tsx)  // confirm routes

RULES
- No new dependencies.
- Use fetch with JSON; handle loading + error states with toasts.
- Dates in UI as dd/MM/yyyy.

BACKEND (confirm endpoints exist & fix if not)
Admin
- GET    /api/admin/lead-forms                       → [{id,title,slug,updatedAt}]
- POST   /api/admin/lead-forms                       → {id,slug} (create default form + 7 default questions)
- GET    /api/admin/lead-forms/:id                   → {form, questions}
- PATCH  /api/admin/lead-forms/:id                   → {ok:true} (upsert form & questions by orderIndex)
- DELETE /api/admin/lead-forms/:id                   → {ok:true} (soft delete optional)
Public
- GET    /api/leads/public/:slug                     → {form, questions}
- POST   /api/leads/public/:slug/submit              → {ok:true, projectId, leadId, afterSubmit:{type,url?,message?}}
Embed helper
- GET    /embed.js                                   → JS as previously spec’d

If any are missing, add them; otherwise leave logic as-is. Ensure express.json() before routes, auth guard only on /api/admin/*.
